# bartlopen Run Coach — Template 🏃‍♂️🏃‍♀️

Herbruikbare basis voor **persoonlijke hardloop-apps** van Coach Bart (@bartlopen).
Elke app is een lichte web-app (PWA): vast trainingsschema, trainingen afvinken &
invullen, Strava-achtige stats, badges en coaching-praatjes. Alles draait lokaal in
de browser, voortgang blijft op het apparaat. Geen server, geen build-stap.

## Zo maak je een app voor een nieuwe loper
Zie **[BUILD.md](BUILD.md)** voor het volledige recept. Kort:
1. Kopieer deze map → nieuwe repo `bartlopen-<naam>`.
2. Pas in `app.js` de blokken **CONFIG → ZONES → PLAN → COACH/WHY/INFO/BADGES** aan.
3. `generate_og.py` voor de social preview, GitHub Pages aan, link delen.

## Twee varianten
- **Afstand-gebaseerd** (km) — voor de meeste lopers. Referentie: Dave, Marinke.
- **Tijd-gebaseerd** (minuten) — fijn bij blessure-opbouw. Referentie: Sietse.
